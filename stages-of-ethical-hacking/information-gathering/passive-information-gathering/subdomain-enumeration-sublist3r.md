# Subdomain Enumeration - Sublist3r



## **Sublist3r**

[GitHub repo](https://github.com/aboul3la/Sublist3r)

## Install (kali repo)

```bash
sudo apt-get install sublist3r
```

## Running the command

```bash
sublist3r -d $DOMAIN_NAME
```

* Using **ACTIVE** scan

```bash
sublist3r -d $DOMAIN_NAME -b -t 26
```
